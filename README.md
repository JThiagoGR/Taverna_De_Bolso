# TAVERNA DE BOLSO - ONLINE

## Deploy em 1 clique:

### Railway.app (RECOMENDADO)
1. Acesse railway.app
2. New Project → Deploy from GitHub
3. Faça upload deste ZIP
4. Pronto! Link: https://seu-app.up.railway.app

### Render.com
1. render.com → New Web Service
2. Upload ZIP
3. Build: npm install
4. Start: node server.js

### Fly.io
fly launch
fly deploy
