# 🍺 Taverna De Bolso

VTT (Virtual Tabletop) para RPG de mesa - otimizado para celular

## 🚀 Instalação Local

```bash
npm install
node server.js
```
Acesse: http://localhost:3000

## 🌐 Deploy Railway

1. Faça upload para GitHub
2. Conecte no Railway.app
3. Deploy automático
4. Gere domínio público

## ✨ Funcionalidades

- ✅ Tokens com HP/CA
- ✅ NPCs
- ✅ Paredes (lápis)
- ✅ Mapa com upload
- ✅ Névoa de guerra
- ✅ Luz dinâmica
- ✅ Régua
- ✅ Dados d20
- ✅ Zoom sincronizado
- ✅ Sincronização em tempo real

## 🎮 Como Usar

**Mestre:** Entre como mestre, carregue mapa, crie NPCs
**Jogadores:** Entrem na mesma sala
